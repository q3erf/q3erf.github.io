## vim多行插入
### 行首插入
```
# 进入视图模式
ctrl+v

# 选择需要插入的多行
j or k

# 跳到行首
shift + 6 即 ^

# 插入
大写I
```

### 行尾插入
```
# 进入视图模式
ctrl+v

# 选择需要插入的多行
j or k

# 跳到行尾
shift + 6 即 $

# 插入
大写A
```
